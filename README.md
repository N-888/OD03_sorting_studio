# Sorting Studio

- современное учебное приложение на Python для изучения алгоритмов сортировки 
- с красивым интерфейсом на `customtkinter`.
- это современное учебное приложение на Python, 
которое наглядно демонстрирует работу классических алгоритмов сортировки в красивом адаптивном интерфейсе. 
- проект сочетает теорию и практику: 
- пользователь может ввести собственный набор чисел, 
- выбрать нужный алгоритм, 
- увидеть итоговый результат, 
- пошаговое выполнение и сравнение нескольких алгоритмов между собой. 
- Интерфейс выполнен в современном стиле с заголовками, иконками, карточками, вкладками и поддержкой смены темы, благодаря чему программа выглядит аккуратно и удобно даже для начинающего пользователя. Проект отлично подходит для учебных работ, демонстрации на уроке, защиты задания и самостоятельного изучения алгоритмов.

## Коротко сильные стороны проекта

- современный визуальный интерфейс;
- удобный запуск в Windows и PyCharm;
- классическая реализация нескольких сортировок;
- подробные пояснения по шагам;
- поддержка сортировки по возрастанию и убыванию;
- сравнение алгоритмов в одном окне;
- встроенная теория;
- понятная структура по папкам и файлам;
- код удобно расширять и улучшать.

## На что обратить внимание, чтобы все запустилось без ошибок

- Папка должна называться нормально, без случайных лишних файлов.
- Папка app должна быть именно папкой.
- Файлы должны называться точно так, как я написала.
- Отступы в Python должны сохраниться.
- Сначала вставьте код, потом установите зависимости.
- Запускайте именно main.py.

## Что умеет проект

- показывает работу нескольких классических алгоритмов сортировки;
- сортирует по возрастанию и по убыванию;
- выводит итоговый результат;
- показывает подробные шаги выполнения;
- сравнивает все алгоритмы между собой по времени и операциям;
- содержит встроенную теорию прямо в приложении;
- аккуратно адаптируется при изменении размера окна.

## Алгоритмы

- пузырьковая сортировка;
- быстрая сортировка;
- сортировка выбором;
- сортировка вставками.
