# Sorting Studio

Современное учебное приложение на Python для изучения алгоритмов сортировки с красивым интерфейсом на `customtkinter`.

## Что умеет проект

- показывает работу нескольких классических алгоритмов сортировки;
- сортирует по возрастанию и по убыванию;
- выводит итоговый результат;
- показывает подробные шаги выполнения;
- сравнивает все алгоритмы между собой по времени и операциям;
- содержит встроенную теорию прямо в приложении;
- аккуратно адаптируется при изменении размера окна.

## Алгоритмы

- пузырьковая сортировка;
- быстрая сортировка;
- сортировка выбором;
- сортировка вставками.
